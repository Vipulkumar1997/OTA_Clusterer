Web Check-In

Last Minute Web Check-In













Bequem und einfach
Mit dem Web Check-In können Sie bequem von Zuhause aus bereits einchecken, einen Sitzplatz auswählen und anschliessend die Bordkarte ausdrucken.
Das Web Check-In ist von 23 Stunden bis zu 1 Stunde vor Abflug möglich und steht den folgenden Passagieren zur Verfügung:
Passagiere, die ohne Kleinkind (Alter bei Abflug 0-2 Jahre) reisen.
Passagiere, die vom Flughafen Zürich und Genf abfliegen.

Und so funktioniert es
Schritt 1:
Halten Sie bitte die Reisedokumente aller Teilnehmer für das Web Check-in bereit, denn einige Staaten verlangen zusätzliche Passagier-Informationen beim Check-in. Im Speziellen sind dies die Nummer Ihres Passes oder der Identitätskarte, Ausstellungsort und Ablaufdatum sowie das Geburtsdatum des einzelnen Fluggastes.
Schritt 2:
Geben Sie Ihren Vor- respektive Nachnamen und die Flugdaten ein, um Ihre Buchung aufzurufen. Falls Ihre Bestätigung bei einzelnen Personen mehrere Vor- oder Nachnamen aufweist, müssen diese jeweils ohne Bindestrich oder Abstand erfasst werden.
Schritt 3:
Wählen/bestätigen Sie den Flug für das Web Check-In.
Schritt 4:
Suchen Sie sich einen Sitz aus und geben Sie an, ob Sie ein oder mehrere Gepäckstücke aufgeben werden.
Schritt 5:
Drucken Sie Ihre Bordkarte aus oder laden Sie diese auf ihr Smartphone.

Nun haben Sie das Web Check-In abgeschlossen. Auf der ausgedruckten Bordkarte finden Sie alle Informationen zum „Baggage drop-off“- Schalter für die Gepäckabgabe, Ihrem genaueren Flugsteig und weitere wichtige Angaben, die Sie benötigen, bevor Sie sich zum Flughafen begeben.
Achtung:
Fluggäste, die mit Tieren reisen, allein reisende Kinder und alle Passagiere, die eine Betreuung benötigen, erhalten eine Check-in Bestätigung, welche dann am Gepäckabgabeschalter gegen eine Bordkarte eingetauscht werden muss.

Hilfe-Center
Falls Sie Schwierigkeiten haben das Web Check-In abzuschliessen oder die Bordkarte auszudrucken, können Sie jeweils die untenstehende Telefonnummer Ihrer Airline an Ihrer Abgangs-Station anrufen. Gerne checken wir Sie auch telefonisch ein.
